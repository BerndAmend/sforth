# Sforth Language Support

Provides syntax highlighting for the `sforth` programming language in `.fs` and
`.sforth` files.

## Features

- Highlighting for strings, template strings
- Highlighting for numbers
- Highlighting for inline javascript block `:[ ... ]:`
- Forth stack comments `( ... )`
- Common stack words `dup`, `swap`, etc.
- Custom keywords mapping

## Build with

```bash
npm run package
```
